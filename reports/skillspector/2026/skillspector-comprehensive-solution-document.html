2. What Is SkillSpector? ↑ top

SkillSpector is NVIDIA's open-source security scanner purpose-built for AI agent skills — specifically LangGraph-based skill ecosystems. It detects vulnerabilities, malicious patterns, and trust-boundary violations in skill definition files (SKILL.md) before those skills are installed and executed by an AI agent runtime.

2.1 Dual-Stage Analysis Engine

StageMethodWhat It CatchesSpeed
Static ASTYARA rules + AST parsing of embedded code blocksMalware signatures, shell injection, obfuscated code, suspicious imports, file-system escapes~1-5s/skill
LLM SemanticMulti-model semantic evaluation of NL instructionsInstruction hijacking, trust-boundary violations, PII leakage, tool misuse, authority override~10-30s/skill

2.2 Scoring Model

ScoreSeverityAction
0–19LOWPass — no blocking
20–54MEDIUMWarning — review recommended
55–99HIGHBlocked by pre-push — remediation required
100CRITICALBlocked — CSO review mandatory

6. Client-Side Git Hooks ↑ top

6.1 pre-push — Blocking Security Gate

.hooks/pre-push → .git/hooks/pre-push

Blocks push if any staged skill file triggers HIGH or CRITICAL findings. Bypass: git push --no-verify (requires documented justification).

6.2 post-merge — Third-Party Validation Gate

.hooks/post-merge → .git/hooks/post-merge

Fires after every merge/pull. Runs full recursive scan to validate external contributions. Does not block — reports warnings.

6.3 Activating on a New Workstation

cp .hooks/pre-push .git/hooks/pre-push
cp .hooks/post-merge .git/hooks/post-merge
chmod +x .git/hooks/pre-push .git/hooks/post-merge

10. Critical Skills Analysis ↑ top

nextflow-development (31 findings)

Bio-research workflow skill. Flagged for embedded shell execution, container orchestration (Docker/Singularity), and pipeline script generation. In HPC environments, these patterns could enable arbitrary code execution on compute clusters.

Recommendation: Baseline after CSO review. Consider runtime sandbox.

docx (28 findings)

Office document generation. Flagged for python-docx execution, template injection, and file-system writes. Primary concern: malicious templates embedding macros or external references.

Recommendation: Baseline after template source audit. Enforce whitelist.

optimo-rcm (11 findings)

RCM analysis pipeline. The 5-perspective engine (8 base + 4 engines + 5 domain packs) triggers on dynamic data ingestion and multi-source synthesis crossing trust boundaries.

Recommendation: Baseline after CSO confirms data pathways.

pptx (26 findings)

PowerPoint generation. python-pptx execution, template manipulation, macro injection surfaces.

Recommendation: Baseline after template audit. Strip active content.

xlsx (30 findings)

Excel generation — highest finding count. Flagged for openpyxl execution, formula injection (CWE-1236), CSV parsing chains. Cells starting with =, +, -, @ can execute commands when opened in Excel.

Recommendation: Implement formula sanitization before baseline. Highest downstream risk to non-technical users.
