## Solution Architecture

### 1. Local Installation
- **SkillSpector v2.3.13** installed via uv from NVIDIA/SkillSpector.git
- GitHub secrets set: `ANTHROPIC_API_KEY` + `SKILLSPECTOR_PROVIDER`

### 2. GitHub Actions CI Pipeline
- Trigger: push/PR to main/develop on skill files
- Cron: every Monday midnight UTC
- Output: SARIF → GitHub Security tab

### 3. Git Hooks
- **pre-push**: Blocks on HIGH/CRITICAL findings
- **post-merge**: Validates third-party skills after merge

### 4. Auto-Sync Script
- `.hooks/update-skillspector.sh` — checks upstream, reinstalls on delta

### 5. CLAUDE.md Policy
- Zero-trust mandate codified in `~/.claude/CLAUDE.md`
- Dual-stage gate required before any skill installation
